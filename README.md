# CHECK-MATE!

Videe-Group-Meeting is a simple video chat application for multi-users based on React, Node Express and WebRTC.

This project is **deployed** at: https://check-mate-btp.herokuapp.com/

**Technologies Used**

- React
- Node + Express
- WebRTC
- Socket.io
- [Syled-components](https://styled-components.com/)
- [Simple-peer](https://github.com/feross/simple-peer)

**Contributors:** [Saurabh Joshi](https://github.com/CodingJoshi)

---

## Features

- Join a room
- Video Streaming
- Text chat
- Mute Video/Audio
- Screen Sharing

## Installation

### Setup
**Client**
> Move to client folder, update and install this package
<pre>
  <code>
    /* Install */
    npm install
    
    /* Run */
    npm start
  </code>
</pre>

**Server**
> Move to server folder, update and install this package
<pre>
  <code>
    /* Install */
    npm install
    
    /* Run */
    npm run dev
  </code>
</pre>
